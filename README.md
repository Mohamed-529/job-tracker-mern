# Job Tracker 🚀

A full-stack MERN application to manage and track job applications efficiently.

## 📌 Overview

Job Tracker helps users keep track of job applications, their status, and progress in one place. It simplifies the job search process by organizing all application data.

---

## ✨ Features

* 🔐 User Authentication (Login / Register)
* ➕ Add new job applications
* 📝 Edit and update job details
* ❌ Delete job entries
* 📊 Track job status (Applied, Interview, Offer, Rejected)
* 🔍 Search and filter jobs
* 📱 Responsive UI

---

## 🛠 Tech Stack

* **Frontend:** React.js, CSS
* **Backend:** Node.js, Express.js
* **Database:** MongoDB
* **Authentication:** JWT (JSON Web Tokens)

---

## 📂 Folder Structure
